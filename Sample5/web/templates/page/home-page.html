This is the home page

This sample is intended to be a starting point for building a new website. You should copy this project, rename Sample5 to the name of your website and replace the content with your website content.

Features

  • Standard header and footer on all pages
  • Master page defines shared page elements
  • Page content defined by HTML templates
  • Styles defined in LESS syntax
  • Support for Vue.js view models
  • Responsive data driven menu
  • Users can create an account then login agian later. Includes password reset etc
  • Favicon support
  • Starting point for packages and modules

To Do

To turn this into your website yo need to

  • Copy this project and check it into source control
  • Check that the website compiles and runs
  • Case sensitive search and replace all occurences of Sample5 and sample5 with the name of your website. You must do this in every file including the csproj file.
  • Check that the website still compiles and runs
  • Modify the html that is output by the PageHeadComponent class
  • Define the menu navigation for your website in the MenuDataProvider clsss
  • Replace this html with the content of your home page
  • Replace the html in the FooterComponent class with the footer text you want or change it to render a template
  • Change main.less to the styling you prefer
  • Add templates for the other static pages on your website under /web/templates/pages
  • Route requests to pages by making a copy of the HomePage class for each page on your website

For more advanced use cases you may also need to

  • Create Vue.js view models by adding .vue and .js files to the /web/templates folder
  • Write some Data Providers then add data binding expressions to your .html templates
  • Localize the text on your website
  • Add AJAX services that can be called from client-side javascript

All of these additional steps are fully supported by the Owin Framework
